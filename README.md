# diveup
Repositorio de desarrollo del juego DiveUp

Requerimientos:
- [SFML 2.5.1](https://www.sfml-dev.org/tutorials/2.5/start-linux.php)
- cmake

Para compilar el juego:
- mkdir build
- cd build && cmake ..
- cd .. && cmake --build build
